# Flappy Wyvern

A retro-inspired reimagining of the classic "Flappy Bird" mechanic, built using the **Godot Engine**. This project replaces the colorful pipes and birds of the original with the dark, pixelated aesthetic of 90's dungeon crawlers.



---

## Project Overview
You control a Wyvern trying to survive an endless gauntlet of hellish statues.

This project was developed using **Godot’s 2D physics engine**, signal management, and high-score persistence, all while paying homage to your favorite 90's dungeon crawlers.

## Features
* **Physics-Based Gameplay:** Fine-tuned gravity and impulse settings for a responsive "flap."
* **Infinite Scrolling:** Procedurally generated obstacles that increase in difficulty.
* **Score System:** Keeps track of your run to know your "glory."

## Built With
* [Godot Engine](https://godotengine.org/) (v4.x)
* GDScript
